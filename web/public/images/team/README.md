# Fotos da equipe

Coloque aqui os retratos usados na seção de equipe e no modal do integrante.

Recomendação prática:
- Formato: `.webp` preferencialmente, ou `.jpg`/`.jpeg`
- Proporção: quadrada, com corte centralizado
- Tamanho: 800x800 px ou maior
- Nome dos arquivos: use o valor definido no campo `photo` do integrante, por exemplo `nathan-borges.webp`
- Fundo: neutro ou coerente com a identidade visual da apresentação

Se a imagem não existir, a interface cai automaticamente para as iniciais do integrante.
